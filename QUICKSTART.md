# Быстрый старт

## 1. Создайте файл .env

Скопируйте `.env.example` в `.env` и заполните токены:

```bash
cp .env.example .env
```

Заполните в `.env`:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```

## 2. Запустите Docker контейнеры

```bash
docker-compose up -d --build
```

## 3. Откройте фронтенд

Откройте в браузере: http://localhost

## 4. Просмотр логов

```bash
docker-compose logs -f
```

## 5. Остановка

```bash
docker-compose down
```

## Структура проекта

- **Фронтенд**: http://localhost (порт 80)
- **API**: http://localhost:8000 (порт 8000)
- **База данных**: `./data/messages.db` (монтируется в контейнер)

## Функции фронтенда

- Просмотр всех сообщений (принятых и отклоненных)
- Кнопки "Принять" / "Отклонить" для каждого сообщения
- Автоматическое обновление каждые 5 секунд
- Ручное изменение статуса модерации
- Влияние статуса на попадание в очередь (только "ok" попадают в очередь)
